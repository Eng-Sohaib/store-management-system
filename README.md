A robust, full-stack web application designed to handle inventory management and point-of-sale (POS) transactions for retail businesses. 

This project demonstrates a clean, decoupled architecture using a RESTful Java Spring Boot backend and a responsive, modern React frontend. It ensures ACID-compliant database transactions to prevent inventory data corruption during concurrent checkouts.

## Key Features

* **Interactive POS Dashboard:** A dynamic checkout interface that calculates totals in real-time and processes transactions securely.
* **Real-time Inventory Tracking:** Stock levels are automatically deducted upon successful checkout using `@Transactional` database rollbacks to ensure data integrity.
* **CRUD Operations:** Full capability to Create, Read, Update, and Delete product listings via a clean UI.
* **Responsive Design:** Styled with Tailwind CSS for a modern, accessible, and highly responsive user experience across all devices.
* **RESTful API Architecture:** Clean separation of concerns between the client and server.

## Technology Stack

**Backend (REST API):**
* **Java 17 / 21**
* **Spring Boot 3+** (Web, Data JPA)
* **Hibernate** (ORM)
* **MySQL 8+** (Relational Database)
* **Lombok** (Boilerplate reduction)

**Frontend (User Interface):**
* **React 18**
* **Vite** (Next-generation frontend tooling)
* **Tailwind CSS v4** (Utility-first styling)
* **Fetch API** (Asynchronous HTTP requests)
